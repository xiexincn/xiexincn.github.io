🚘 Reliability Analysis of In-vehicle Network

  • Vehicle Ethernet: Automotive Ethernet In-vehicle Network provides flexibility and freedom in the implementation of the in-vehicle EE architecture, which provides high bandwidth at low cost. However, the unrelibility performance of Ethernet communication become the major concerns of its adapation.
  • DES: we uses the diseceret event simulation to build a digital replica of the in-vechicle network, which can be leveraged to efficiently test the network performance under varying network configurations and traffic models.
  • DNC: we also leverage the network calculs to analyze the performance bound of the system, which helps to understand the wrost case delay bound and memory useage of the system.
  • Objective: design an effective tool to valid and capture the potential risks in in-vehicle network communication.
